# Upload Rules Reference

## Complaint submission options

### Option 1
- write complaint text directly
- optional evidence file

### Option 2
- upload one complaint document
- optional evidence file

## Hard limits

- max 1 complaint document
- max 1 evidence document
- max 2 MB per file

## Recommended allowed file types

- PDF
- DOC
- DOCX
- JPG
- PNG

## Validation rules

A complaint must have at least one of:
- text description
- complaint document

Both frontend and backend must validate file rules.
Frontend validation improves UX.
Backend validation enforces policy.
